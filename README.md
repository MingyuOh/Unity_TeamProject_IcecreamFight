## 📝 개요
<img src="https://user-images.githubusercontent.com/16281218/219337635-9cb46c60-1be7-460c-8676-d40ed149814d.png" width="800" height="600"/>    

> - 프로젝트: Unity를 이용하여 아이스크림을 던져 상대방(멀티플레이) 또는 AI(싱글플레이)와 대전하는 3D 게임
> - 기획 및 제작: 오민규 외 1명(클라이언트 1명)    
> - 분류: 팀 프로젝트   
> - 담당 업무: 게임 코어 개발, 클라이언트, 서버, 데이터베이스    
> - 제작 기간: 60일    
> - 게임 소개:   
> &emsp;3D 멀티플레이 게임으로 아이스크림을 무기로 하여 상대방과 대전하는 게임이다.   
> &emsp;어렸을 적 밖에서 눈싸움을 하며 놀았던 기억으로부터 게임 규칙을 결정하였다.        
> &emsp;눈싸움에서 사용하는 무기를 스노우볼 대신 아이스크림으로 대체하였으며,     
> &emsp;천진난만한 아이들이 아이스크림을 통해 장난치는 모습을 연상하여 게임을 기획하였다.    
> &emsp;기존의 게임에서 성행했던 폭력성, 선정성같이 게임 속에서 발견할 수 있는 부정적인 요소들을 대체하기      
> &emsp;위해 귀여운 동물 캐릭터를 주인공 캐릭터로 사용하고, 아이스크림을 주 무기로 사용하였다.    
> &emsp;정해진 시간 안에 상대방을 제압한 횟수가 높은 플레이어가 승리하는 게임이다.    

## 🏆 목표      
- 해당 프로젝트는 창의 학기제 과제로 실제 서비스를 목표로 게임을 개발하는 것이다.    
- 게임 개발 이외에도 창업 활동(사업 계획서 작성, 창업 경진 대회 참여, 청년 창업 정부 지원 등록 등)을 하여 예비창업가로서의 역량을 향상시킨다.    
- 다양한 서비스를 프로젝트에 적용함으로써 API 사용 경험을 늘리고, 멀티플레이를 가능하게 하여 네트워크 게임의 전체 프로세스를 경험하고 개발 역량을 향상하기 위해 진행하는 프로젝트이다.
